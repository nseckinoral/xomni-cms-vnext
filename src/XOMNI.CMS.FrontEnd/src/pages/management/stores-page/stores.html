Stores

Stores define physical locations where your in-store devices reside. The relationship between a device and a store is defined through the license attached to a device.

Some of the scenarios stores are used for are;

  • GPS based store search.
  • Omni-Discovery in a store using location.
  • Omni-Discovery in a store using license.
  • In-Store Metadata APIs to enable store based item data.
  • Store Storage APIs for shared storage across devices.
